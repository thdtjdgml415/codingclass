HTML 기본 문서

HTML 기본문서의 <html> 요소는 <head> 요소와 <body> 요소로 구성되어 있으며, <meta> 요소 안에는 meta, title, link, style, script등의 요소들이 올 수 있습니다.

요소명 설명
<meta> HTML 문서의 설명, 키워드, 문서의 작성자 등을 지정합니다.
<title> HTML 문서의 제목을 지정합니다.
<link> 외부의 CSS파일을 HTML문서에 연결할 때 사용합니다.
<style> HTML 문서 내에서 스타일 (CSS)의 정의합니다.
<script> 자바스크립트를 HTML 문서 내부에서 정의하거나 외부에서 자바스크립트 파일을 불러올 때 사용합니다.
lang 속성은 HTML 문서에서 꼭 정의해야 하는 속성

<i lang="de">Ich liebe dixh</i>
