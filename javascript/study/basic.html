자바스크립트를 사용해 알고리즘 문제를 해결하는 것입니다.

자바스크립트란?

JavaScript는 웹 페이지에서 복잡한 기능을 구현할 수 있도록 하는 스크립트 언어 또는 프로그래밍 언어입니다. HTML은 정적인 레이아웃을 표현한다면, 자바스크립트는 동적인 기능을 표현합니다.

출력하는 방법

자바스크립트에서는 출력할 수 있는 방법이 여러가지가 있습니다.

분류 메서드
입력 prompt()
출력 document.write
alert()
console. log()
확인 confirm()

자바스크립트 사용 방법

자바스크립트의 사용방법
1. 외부파일 연결하는 방법
2. 스크립트 태그에 사용하는 방법
3. 태그에 직접 기술하는 방법등이 있습니다.


    function solution(absolutes, signs) {
        let k = 0;
          absolutes.forEach((i, el) => {
              if(signs[el] === true) {
                  k += absolutes[el];
              } else {
                  k -= absolutes[el];
              } 
          })
          return k
      }
