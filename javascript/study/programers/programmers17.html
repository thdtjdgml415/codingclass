문자열 내 p와 y의 개수

설명

문자열 s의 길이 : 50 이하의 자연수
문자열 s는 알파벳으로만 이루어져 있습니다.


제한 사항

n은 10,000,000,000이하인 자연수입니다.

s answer
"pPoooyY" true
"Pyy" false

정답 풀이

function solution(s){
    let x = [...s]
    let p =0;
    let y =0;
    let z =0;
    for(let i = 0; i<=x.length; i++) {
        if(x[i] == 'p' || x[i]=='P') {
            p++
        } else if (x[i]=='y'||x[i]=='Y'){
            y++
        } else {
            z++
        }
    }
    if(p==y) return true
    else return false
}

해설

이번 문제는 노가다 였습니다. 하지만 이번엔 이전 문제에서 사용한
스프레드 연산자를 다시 사용해보는 시간이였습니다. 스프레드 연산자를
사용하여 배열로 바꾸고 for문을 사용하여 배열 x의 길이만큼
반복합니다. 이때 조건문에서 "p", "P"와 같으면 변수 p의 카운트를 1씩
증가시키고 "y", "Y"이면 변수 y의 카운트를 1씩 증가시킵니다. 그러고
변수 p와 y의 값이 같으면 true를 반환하고 틀리면 false를 반환하도록
하였습니다. 좀 노가다였지만 그래도 풀었네요 이제 이정도는 풉니다 ㅎ

function numPY(s) {
    return s.match(/p/ig).length == s.match(/y/ig).length;
