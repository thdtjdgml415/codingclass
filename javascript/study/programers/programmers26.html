부족한 금액 계산하기

설명

새로 생긴 놀이기구는 인기가 매우 많아 줄이 끊이질 않습니다. 이
놀이기구의 원래 이용료는 price원 인데, 놀이기구를 N 번 째 이용한다면
원래 이용료의 N배를 받기로 하였습니다. 즉, 처음 이용료가 100이었다면
2번째에는 200, 3번째에는 300으로 요금이 인상됩니다. 놀이기구를 count번
타게 되면 현재 자신이 가지고 있는 금액에서 얼마가 모자라는지를 return
하도록 solution 함수를 완성하세요. 단, 금액이 부족하지 않으면 0을
return 하세요.

제한 사항

놀이기구의 이용료 price : 1 ≤ price ≤ 2,500, price는 자연수
처음 가지고 있던 금액 money : 1 ≤ money ≤ 1,000,000,000, money는 자연수
놀이기구의 이용 횟수 count : 1 ≤ count ≤ 2,500, count는 자연수

입출력 예

price money count result
3 20 4 10

정답 풀이

function solution(price, money, count) {
    let totalPrice = 0;
    
    for(let i=1;i<=count;i++){
        totalPrice += price * i;
    }
    
    return money > totalPrice ? 0 : totalPrice-money
}

해설

이번 문제는 해석이 어려운 문제였습니다. 쉽지 않네요.. 놀이기구 이용료는 price입니다.
count번 놀이기구를 타고 money는 처음 가지고 있던 금액입니다.
놀이기구를 count번 탔을 때 가격을 변수에 담기위해 totalPrice를
선언하고 반복문을 사용하여 count번 놀이기구를 이용할 때까지
반복시켜주도록 합니다. 계속 중첩해서 가격을 지불해야하기 때문에 +=을
사용합니다. 그리고 삼항연산자를 사용하여 놀이기구를 count번 탔을때
내가 지불해야 하는 돈 totalPrice가 처음 가지고 있던 돈 money보다
적다면 0을 반환하고 많다면 내가 지불해야하는 돈 totalPrice가 내가
처음에 가지고 있던 돈 money보다 많다면 totalPrice-money를 하여
부족한 돈을 반환합니다.
