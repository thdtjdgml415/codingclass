약수의 합

설명

정수 n을 입력받아 n의 약수를 모두 더한 값을 리턴하는 함수, solution을
완성해주세요.

제한 사항

n은 0 이상 3000이하인 정수입니다

n return
12 5
28 6

풀이

function solution(n) {
    let a = n;
    let num = 0;
    for(let i =1; i<=n; i++) {
        if(a%i==0){
            num += i;
        }else {
            num += 0;
        }
    }
    return num;
}

해설

매개변수 n을 변수 a에 저장합니다. 그리고 반복문을 사용해서 0은
약수일 수 없기 때문에 i는 1부터 시작하도록 합니다 또한 약수는
나눠지는 수 보다 클 수 없기 때문에 n까지만 반복하도록 합니다. 약수로
나누면 나머지가 0으로 나누어 떨어지기 때문에 a를 i로 나눴을 때
나누어 떨어지면 약수 입니다. 그때 변수 num에 계속 중첩하여
더해줍니다. 나누어 떨어지지 않는 경우 0을 더해줘서 수에 변화가
없도록 하여 num을 반환시킵니다.
