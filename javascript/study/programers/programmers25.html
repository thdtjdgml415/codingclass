문자열 내림차순으로 배치하기

설명

문자열 s에 나타나는 문자를 큰것부터 작은 순으로 정렬해 새로운 문자열을 리턴하는 함수, solution을 완성해주세요. s는 영문 대소문자로만 구성되어 있으며, 대문자는 소문자보다 작은 것으로 간주합니다.

제한 사항

str은 길이 1 이상인 문자열입니다.

입출력 예

s return
"Zbcdefg" "gfedcbZ"

정답 풀이

function solution(s) {
    return [...s].sort().reverse().join("")  
}

해설

이번문제는 문자열을 정렬하는 방법에 대한 문제로 대소문자를 구분하여
정렬하는 방법에 대해 고민을 해보았는데 사실 문자열을 배열로 만들어
정렬을 한다면 대문자는 유니코드 상에서 소문자보다 크기때문에 우선
sort에 아무 함수도 사용하지 않고 오름차순으로 정리합니다. 그리고
배열을 뒤집는 reverse 메소드를 사용하여 거꾸로 정렬하고 join
메소드를 사용해 문자열로 다시 합치면 끝입니다.

다른사람 풀이

function solution(s) {
    return s.split('').sort((val1,val2) => val2.charCodeAt() -val1.charCodeAt()).join('');
}

해설

이 풀이도 sort를 이용하여 정렬한 문제입니다. 하지만 charCodeAt이라는 내장메소드를 사용하여 문자를 유니코드로 변환시켜서 내림차순 정리를 하였는데요 유니코드로 변환시키지 않으면 sort로 문자열을 오름차순으로만 배열할 수 있습니다. 하지만 여기서는 유니코드를 사용해 바로 내림차순으로 정렬하였습니다.
