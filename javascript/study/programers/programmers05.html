짝수와 홀수

설명

정수 num이 짝수일 경우 "Even"을 반환하고 홀수인 경우 "Odd"를 반환하는
함수, solution을 완성해주세요.

제한 사항

num은 int 범위의 정수입니다.
0은 짝수입니다.

num return
3 "Odd"
4 "Even"

풀이

function solution(num) {
let a = (num%2==0)?  "Even" : "Odd"
return a;
}

해설

짝수일 때 Even 을 출력하고 홀수 일 때 Odd를 출력하는 문제 입니다.
삼항연산자를 사용해서 작업하였습니다. 짝수 일때는 2로 나눴을때
나머지가 0이기 때문에 true일때 Even을 출력하고 나머지 경우는 모두
홀수 이기 때문에 false일 경우 Odd를 출력하게 하였습니다.
