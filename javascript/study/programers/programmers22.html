x만큼 간격이 있는 n개의 숫자

설명

함수 solution은 정수 x와 자연수 n을 입력 받아, x부터 시작해 x씩
증가하는 숫자를 n개 지니는 리스트를 리턴해야 합니다. 다음 제한 조건을
보고, 조건을 만족하는 함수, solution을 완성해주세요.

제한 사항

x는 -10000000 이상, 10000000 이하인 정수입니다.
n은 1000 이하인 자연수입니다.

입출력 예

x n answer
2 5 [2,4,6,8,10]
4 3 [4,8,12]
-4 2 [-4, -8]

정답 풀이

function solution(x, n) {
    return Array.from(Array(n), (n, index) => x * (index + 1));
}

해설

Array(n)은 길이가 n인 유사 배열을 만들고 기존의 배열 값에서 n과
index값을 매개변수로 받고 만들 배열의 조건을 사용하여 원하는 배열을
반환시켰습니다.
