1부터 100까지 출력하기

1.1 while문을 이용해서 1부터 100까지 출력하시요!

let num =1;

while(num <=100){
    document.write(num);
    num++;
}

1.2 for문을 이용해서 1부터 100까지 출력하시요!

for(let num=1; num < 100; num++ ){
    document.write("실행되었습니다1.");
}

1.3 for문을 이용해서 1부터 100까지 출력하시요(홀수, 짝수)

for (let i = 2; i <= 100; i++) {
    if (i % 2 == 0) {
      alert( i );
    }
  }

1.4 for문을 이용해서 1부터 100까지 합을 출력하시요

let sum = 0;
for(let i=1; i<=100; i++){
    sum += i;
}
document.write(sum);
