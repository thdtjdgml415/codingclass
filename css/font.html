font-size, 단위의 고찰

1.px
해상도에 따라 상대적으로 달라지는 기본 단위로서, 다음과 같이 표현합니다. 

                font-size: 12px;
2.%
부모 요소의 글자 크기를 100%기준으로 계산한 %단위로서, 다음과 같이 표현합니다. 

                font-size: 150%;
3.em
부모 요소 의 글자 크기를 100% 기준으로 계산한 100 분의1 단위로서, 다음과 같이 표현합니다. 

                font-size: 1.5em;
4.rem
rem은 em과 비슷하지만 부모 요소가 아닌 최상위 요소의 크기를 100% 기준으로 계산합니다.rem의 r 은
root 즉, html 요소에서 지정한 글자 크기가 1rem이 됩니다. 

                font-size: 150%;
5.vw,vh
vw는 뷰포트 너비값의 100분의 1 단위, vh는 뷰포트 높이값의 100분의 1 단위이다. 

                font-size: 10vw;
